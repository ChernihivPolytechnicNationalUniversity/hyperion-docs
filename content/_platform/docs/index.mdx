## Getting Started

1. **Choose the right document type** - Review [Documentation Types](/docs/platform/guides/documentation-types) to pick the appropriate format
2. **Use a template** - Start from a [template](/docs/platform/templates) to ensure consistency
3. **Follow writing standards** - Check the [Writing Guide](/docs/platform/guides/writing-guide) for style conventions
4. **Add diagrams** - Use the [Diagram Guide](/docs/platform/guides/diagram-guide) to visualize architecture
