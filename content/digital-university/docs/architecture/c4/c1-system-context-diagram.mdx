## System Boundary

**NPP Portal** (University Faculty Work Portal) is a web application that manages faculty profiles and tracks their professional activities across scientific, methodological, and organizational categories. It calculates performance ratings and provides statistics for departments and individuals.

## Users

| Actor             | Role                    | Primary Use Cases                                                      |
| ----------------- | ----------------------- | ---------------------------------------------------------------------- |
| **Guest**         | Unauthenticated visitor | View public faculty directory, search teachers by name/department      |
| **Teacher**       | Faculty member          | Manage own profile, submit work activities, view personal statistics   |
| **Director**      | Department head / Dean  | View department staff, review activities, access department statistics |
| **Administrator** | System admin            | Manage users, organizational structure, dictionaries, system settings  |

## External Systems

| System                  | Integration                 | Status  |
| ----------------------- | --------------------------- | ------- |
| **Notification System** | SMTP for password reset     | Planned |
| **Export System**       | Excel/PDF report generation | Planned |
| **University LDAP**     | SSO authentication          | Future  |

## Data Flows

1. **Public Access**: Guests can view non-sensitive faculty information without authentication
2. **Authenticated Access**: Teachers, Directors, and Admins authenticate via JWT tokens
3. **Report Export**: Statistics and activity lists can be exported to Excel/PDF
4. **Email Integration**: System sends password setup/reset emails
